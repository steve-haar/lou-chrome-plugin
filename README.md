# Lou Barcode Label Generator

### [&#x2B07; Click here to install](https://steve-haar.github.io/lou-chrome-plugin)

A Chrome extension for [Lou by Evosus](https://lou.evosus.com) that adds a **Print Label** button to SKU profile pages. Click it to generate a scannable Code 128 barcode label and print it on Avery label sheets.

## Features

- Reads UPC, description, and retail price directly from the page
- Generates a Code 128 barcode
- Prints a full Avery sheet (supports 5160, 5161, 5163, 5164, 5168, 5165)
- Remembers your last-used Avery template

## Installation

Chrome does not require any account or payment to install an unpacked extension for personal use.

1. **Download the extension**
   - Click the green **Code** button on this page → **Download ZIP**
   - Unzip the file anywhere on your computer (e.g. your Desktop)

2. **Open Chrome Extensions**
   - Go to `chrome://extensions` in your browser
   - Enable **Developer mode** using the toggle in the top-right corner

3. **Load the extension**
   - Click **Load unpacked**
   - Select the unzipped folder

4. **Done** — navigate to a SKU page on lou.evosus.com and look for the **Print Label** button alongside Edit SKU, Images, etc.

## Usage

1. Open a product/SKU page in Lou
2. Click **Print Label** in the SKU profile card grid
3. Choose your Avery template from the dropdown
4. Click **Print Label** — a full sheet of labels opens and the print dialog appears automatically

## Supported Avery Templates

| Template | Size | Labels/Sheet |
|----------|------|-------------|
| 5160 | 2⅝" × 1" | 30 |
| 5161 | 4" × 1" | 20 |
| 5163 | 4" × 2" | 10 |
| 5164 | 4" × 3⅓" | 6 |
| 5168 | 5" × 3½" | 4 |
| 5165 | Full sheet | 1 |
